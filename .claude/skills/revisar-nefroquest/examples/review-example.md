# Exemplo de revisão

Este exemplo demonstra estrutura e nível de crítica. Os dados bibliográficos são fictícios e não devem ser reutilizados.

# Veredito

**Revisão maior**

**Equivalente legado (auxiliar):** REESCREVER

**Confiança:** alta

**Evidência:** VERIFICADA

**Pendência:** NENHUMA

**Autorização de publicação:** BLOQUEADA — o veredito não é de aprovação (Handbook §18.4, Regra 7c); mesmo com Evidência VERIFICADA e Pendência NENHUMA, a publicação só seria LIBERADA se o veredito fosse "aprovada" ou "aprovada com pequenos ajustes" após a correção abaixo.

# Justificativa

A competência clínica é relevante, mas o enunciado omite um dado que altera a conduta e duas alternativas permanecem defensáveis. A referência citada é uma revisão genérica, embora exista diretriz específica.

# Problemas encontrados

1. **CRÍTICO — duas respostas defensáveis:** a ausência do estado hemodinâmico permite escolher A ou C.
2. **MAIOR — fonte inadequada:** a referência não é específica para a recomendação cobrada.
3. **MAIOR — explicação incompleta:** não diferencia indicação imediata de estratégia expectante.
4. **MENOR — sigla não definida:** a primeira ocorrência deve trazer o termo por extenso.

# Texto para o Antigravity

**SUBSTITUIR A QUESTÃO POR:**

## Pergunta

Paciente de [idade] com [cenário], apresenta [dados decisivos]. Encontra-se [informação que elimina ambiguidade]. Qual é a conduta mais apropriada neste momento?

## Alternativas

**A**  
Conduta plausível, mas inadequada pelo momento clínico.

**B**  
Conduta correta e completa.

**C**  
Conduta aplicável a outro cenário claramente distinguível.

**D**  
Conduta baseada em erro clínico frequente.

## Resposta correta

**B — Conduta correta e completa.**

## Explicação

A alternativa B é a melhor resposta porque [dados do caso] caracterizam [situação] e sustentam [conduta]. A alternativa A seria apropriada apenas se [...]. A alternativa C confunde [...]. A alternativa D não é indicada porque [...]. Na prática, deve-se [...], com monitorização de [...]. A recomendação se aplica a [população] e não deve ser extrapolada para [...].
