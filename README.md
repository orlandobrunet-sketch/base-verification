# NefroQuest Pro (Flashcards de Nefrologia)

App HTML simples para treino em nefrologia com progressão estilo RPG.

## Como abrir (modo fácil)
1. Baixe ou copie a pasta do projeto.
2. Dê **duplo clique** no arquivo `index.html`.
3. O jogo abre direto no navegador.

## Como abrir (modo recomendado)
Se quiser abrir como um site local:

```bash
python3 -m http.server 8000
```

Depois abra no navegador:

- http://localhost:8000

## Regras do jogo
- 4 alternativas por pergunta.
- Dificuldade aumenta conforme você acerta.
- Você tem 3 erros no total.
- O jogo termina ao gastar as 3 vidas.
- O recorde fica salvo no navegador.
